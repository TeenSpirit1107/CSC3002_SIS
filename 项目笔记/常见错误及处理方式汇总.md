## 2. 面向对象

### 2.1. 函数或类在需要被用到的地方显示未声明 

- 解决方案：forward declaration 
  - 对于函数：……
  - 对于类：位置在#include ... 之下，定义之上。
- 举例：`staff.cpp`中，`course.hpp`和`staff.hpp`相互包含，而Staff constructor时要用到vector<Course>, 于是报错。
  - 解决：
  ```cpp
  // staff.cpp
  
  # include "staff.hpp"
  # include "course.hpp"
  ...
  
  // forward declaration
  class Course;
  
  Course::Course(){
    ...
  }
  //
  ```
